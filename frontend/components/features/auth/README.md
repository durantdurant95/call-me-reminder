# Auth Components

This folder will contain authentication-related components that can be extracted from the auth pages.

## Planned Components

- `login-form.tsx` - Login form component (to be extracted from `app/login/page.tsx`)
- `signup-form.tsx` - Signup form component (to be extracted from `app/signup/page.tsx`)

These components can be extracted to make the page files cleaner and the forms more reusable.
